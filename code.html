                        // get values from the page
                        // start or controller function
                        function getValues() {
                            let startValue = document.getElementById("startValue").value;
                            let endValue = document.getElementById("endValue").value;
                          
                            //validate our input (parse then check)
                            //will make sure numbers are looked at as integers
                            startValue = parseInt(startValue);
                            endValue = parseInt(endValue)
                        
                            if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                                 //call generateNumbers
                                let numbers = generateNumbers(startValue, endValue);
                                //call displayNumbers
                                displayNumbers(numbers);
                            } else {
                                alert("You must enter numbers")
                            }   
                        }
                        
                        // generate numbers from startValue to endValue
                        // logic function
                        function generateNumbers(sValue, eValue){
                            let numbers = []; //empty array

                            // we want to get ALL numbers from start to end
                            for(let i=sValue; i<=eValue; i++){
                                numbers.push(i);
                            }
                            
                            return numbers;
                        }
                        
                        
                        
                        
                        // display numbers - even numbers should be bold
                        // display or view function
                        function displayNumbers(numbers) {
                            let templateRows= "";
                        
                                for (let index = 0; index < numbers.length; index++) {
                                    let className = "even";
                        
                                    let number = numbers[index];
                        
                                    //check to see if number is even
                                    if(number % 2 == 0) {
                                        className="even";
                                    } else {
                                        className="odd";
                                    }
                                    templateRows += `<tr><td class="${className}" >${number}</td></tr>                                }
                                document.getElementById("results").innerHTML = templateRows;
                        }

Write a program that displays a user given number range from 1-100. Return all even numbers in BOLD.

1. Retrieve user input numbers

The getValues() function is doing just that. Using the getElementById method, I can access the user inputted start number (startValue) and end number (endValue). The startValue and endValue are used as parameters for the generateNumbers function. We pass in the user inputted start number and end number. *An input will bring back numbers/intergers as a string...which isn't what we want to be using. We have to validate our inputs to make sure they are integers. So, parseInt() method is used to convert the string into an integer. To continue the validation we use an If/Else Statement and Number.isInteger() method to check if the user has inputted integers or not. If the user did put in integers...great!! - the program will continue to run by calling the generateNumbers() function. If the user did not enter integers, they will be met with an alert and asked to do so.

2. Generate the numbers within the user given range

The generateNumbers() function takes two arguements: sValue and eValue. An empty array is created to hold all the numbers generated. A loop is written to go over all the numbers from the starting number until it hits the end number...adding one number to the list everytime the loop is gone through. Numbers are added to the array by using .push.

3. Display the numbers. Even numbers should be bold.

The displayNumbers() function takes in the generated array of numbers as an argument and displays them on the page for the user. A for loop is used to go through the array of numbers to check and see if the number is even or odd using the modulo operator %. The number is divided by two and if there is no remainder then the number is determined to be even. If the number is even - the class name is set to "even". If the number is "odd" - the class name is set to "odd". Table rows are created with each table cell containing the number with the appropriate class name attached to it and is added to the templateRows string. The class names have been styled in site.css - so the even numbers will be styled with bold text. The innerHTML property is set with the ID of "results" which will have the generated numbers displayed in the results element of the webpage. *The innerHTML property allows you to dynamically change the content of HTML on a webpage using Javascript.
